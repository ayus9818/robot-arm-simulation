# Robot Arm Simulation 🤖

## Description
A 3D robot arm simulation built using Three.js, allowing interactive control of joints and object manipulation in a virtual environment.

## Features
- Real-time 3D rendering using Three.js
- Multiple joint control (base, shoulder, elbow, wrist, claw)
- Object grabbing and release system
- Secondary camera view from the claw
- Keyboard-based interaction

## Controls
- Q / E → rotate base
- W / S → move shoulder
- A / D → move elbow
- R / F → rotate wrist
- T / G → extend/retract arm
- Z / X → open/close claw
- Space → grab/release object

## Tech Stack
- JavaScript
- Three.js
- Vite
- WebGL

## What I learned

- Working with Three.js and WebGL
- Real-time interaction and animation
